# Subaru_Auto_Roational_Idle
A mod for Subaru using Dimemod ECU flash to have Rotational Idle 

code  - has sketch to load to nano

media  - contains pictures to assist with where you need to solder to climate control as well as a video demoing how to use the mod. 

tune settings - Has parameters you need to copy to get Rotational Idle working. Adjust Req Tq to in order to reach your target rotational idle speed. Try to target 1500 RPM for best results. 

wiring  - contains how to connect your wires
